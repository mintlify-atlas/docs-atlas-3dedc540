---
title: "Get Public Timeline"
api: "GET /api/v1/projects/{id}/timeline"
description: "Retrieve approved updates grouped by day for public display"
---

## Authentication

No authentication required. This endpoint returns only approved updates and is publicly accessible.

## Path Parameters

<ParamField path="id" type="string" required>
  Project ID (format: `proj_*`)
</ParamField>

## Response

<ResponseField name="timeline" type="array">
  Array of daily aggregates, sorted by date (newest first)
  
  <Expandable title="Daily aggregate properties">
    <ResponseField name="date" type="string">
      UTC date in YYYY-MM-DD format
    </ResponseField>
    
    <ResponseField name="count" type="number">
      Number of updates on this date
    </ResponseField>
    
    <ResponseField name="entries" type="array">
      Array of enriched update objects (same structure as draft objects, with status="approved")
    </ResponseField>
    
    <ResponseField name="stats" type="object">
      Aggregated statistics for the day
      
      <Expandable title="Stats properties">
        <ResponseField name="files_changed" type="number">
          Total files changed across all updates
        </ResponseField>
        
        <ResponseField name="insertions" type="number">
          Total lines inserted
        </ResponseField>
        
        <ResponseField name="deletions" type="number">
          Total lines deleted
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl -X GET https://buildstory-cli.vercel.app/api/v1/projects/proj_abc123/timeline
```
</RequestExample>
