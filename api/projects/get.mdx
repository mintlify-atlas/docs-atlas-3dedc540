---
title: "Get Project"
api: "GET /api/v1/projects/{id}"
description: "Retrieve project details and pipeline statistics"
---

## Authentication

Accepts either:
- Bearer token via `Authorization: Bearer <token>` header
- Supabase session cookie (for dashboard requests)

## Path Parameters

<ParamField path="id" type="string" required>
  Project ID (e.g., `proj_1a2b3c4d5e6f7890`)
</ParamField>

## Response

Returns project record with additional `stats` object.

<ResponseField name="id" type="string">
  Project ID
</ResponseField>

<ResponseField name="name" type="string">
  Project name
</ResponseField>

<ResponseField name="description" type="string">
  Project description
</ResponseField>

<ResponseField name="owner_id" type="string">
  User ID of project owner
</ResponseField>

<ResponseField name="context" type="object | null">
  Repository context data (if provided during creation)
</ResponseField>

<ResponseField name="overview" type="string | null">
  AI-generated project overview (1-3 sentences)
</ResponseField>

<ResponseField name="overview_generated_at" type="string | null">
  Timestamp of last overview generation
</ResponseField>

<ResponseField name="overview_context_hash" type="string | null">
  Hash of context used for overview generation
</ResponseField>

<ResponseField name="overview_edited" type="boolean">
  Whether owner has manually edited the overview
</ResponseField>

<ResponseField name="created_at" type="string">
  Project creation timestamp
</ResponseField>

<ResponseField name="stats" type="object">
  Pipeline statistics object
  
  <Expandable title="Stats properties">
    <ResponseField name="total_updates" type="number">
      Count of approved enriched updates
    </ResponseField>
    
    <ResponseField name="pending_drafts" type="number">
      Count of draft updates awaiting review
    </ResponseField>
    
    <ResponseField name="total_entries" type="number">
      Total raw commit entries received
    </ResponseField>
    
    <ResponseField name="enriched_entries" type="number">
      Number of entries that have been AI-enriched
    </ResponseField>
    
    <ResponseField name="unenriched_entries" type="number">
      Number of entries still awaiting enrichment (calculated as `total_entries - enriched_entries`)
    </ResponseField>
    
    <ResponseField name="last_received_at" type="string | null">
      Timestamp of most recently received commit
    </ResponseField>
    
    <ResponseField name="last_commit_hash" type="string | null">
      Git hash of most recently received commit
    </ResponseField>
  </Expandable>
</ResponseField>

## Ownership Verification

- Authenticated user must be the project owner
- Returns 404 (not 403) if project exists but user is not the owner
- Legacy API tokens (without user association) skip ownership check
- Used by CLI `buildstory status` command to verify project health
